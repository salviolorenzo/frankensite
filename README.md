# Frankesite

This site is a mash up of three portfolio sites. Cloning various features to learn CSS.

## Features!

    1.Header:
        branding/logo/stylized name
        navigation bar

    2.Hero image:
        poster image on the landing of the site 
        text in the center - short and sweet
        title + short mission statement (less than 10 words)
    
    3.About me:
        quick bio:
            why, how, goals, passions
        photo
    
    4.Projects:
        "cards":
            image/gif of project landing page
            information-dense/ not overwhelming
